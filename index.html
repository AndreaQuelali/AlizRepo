Comandos Git

Inicializar repositorio

git init

Crea un nuevo repositorio Git local en el directorio actual.

Clonar repositorio

git clone <url>

Copia un repositorio remoto en tu máquina local.

Agregar cambios

git add .

Agrega todos los archivos modificados al área de preparación (staging).

Confirmar cambios

git commit -m "mensaje"

Confirma los cambios añadidos al staging con un mensaje descriptivo.

Ver estado

git status

Muestra los archivos modificados y su estado respecto al repositorio.

Ver historial

git log

Muestra un historial de confirmaciones del repositorio.

Subir cambios

git push

Envía tus commits locales al repositorio remoto.

Obtener cambios

git pull

Descarga y combina los cambios del repositorio remoto.

Cambiar de rama

git checkout <rama>

Cambia a otra rama existente en el repositorio.

Crear nueva rama

git branch <nombre>

Crea una nueva rama sin cambiarte a ella automáticamente.

Fusionar ramas

git merge <rama>

Combina una rama con la rama actual.

Eliminar rama

git branch -d <rama>

Elimina una rama local de manera segura.
